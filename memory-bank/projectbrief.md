# Project Brief

## What

**EndlessWorlds** — the full build of **endlessworlds.xyz**, a modern, high-converting consulting website for **EndlessWorlds, LLC**. Two workstreams:

1. **Public Site** — brand identity, service pages, thought-leadership blog, case studies, contact / lead-gen forms, SEO.
2. **Secure Portal** — authenticated internal utilities: dashboard, timesheet tracking, invoicing/billing, client-facing project showcase. Auth via OAuth/JWT; database-backed; potential payment integration.

> Note: the EndlessWorlds, LLC entity is not yet formally approved — we proceed as if it is.

## Goals

- Win consulting clients with a fast, polished, credible public presence + lead capture.
- Run the consultancy's internal operations (time tracking, invoicing) through a secure portal.
- Keep the codebase production-grade and extensible across both workstreams.
